# 📱 Al-Wake — تحويل إلى APK

## الخطوات (5 دقائق فقط)

### 1️⃣ أنشئ Repository جديد على GitHub
- اذهب إلى: https://github.com/new
- اسم المشروع: `al-wake`
- اختر: **Public**
- اضغط: **Create repository**

### 2️⃣ ارفع الملفات
بعد إنشاء الـ repo، اضغط **"uploading an existing file"** وارفع:
```
📁 المجلد كاملاً يحتوي على:
  ├── src/
  │   └── index.html
  ├── .github/
  │   └── workflows/
  │       └── build-apk.yml
  ├── package.json
  └── capacitor.config.json
```

### 3️⃣ انتظر البناء التلقائي
- بعد الرفع، اذهب إلى تبويب **Actions** في الـ repo
- ستجد workflow اسمه **"Build Al-Wake APK"** يعمل تلقائياً
- انتظر ~5-8 دقائق حتى يكتمل ✅

### 4️⃣ حمّل الـ APK
- اضغط على الـ workflow بعد اكتماله
- في الأسفل ابحث عن قسم **"Artifacts"**
- اضغط على **"AlWake-debug-APK"** لتحميل الملف

### 5️⃣ ثبّت على هاتفك
- انقل ملف الـ APK للهاتف
- فعّل "تثبيت من مصادر غير معروفة" من الإعدادات
- ثبّت وافتح التطبيق 🎉
